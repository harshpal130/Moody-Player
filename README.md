# 🎧 Moody Player

A full-stack AI-powered music recommendation app that detects your facial expression and suggests songs based on your mood.

---

## 🚀 Live Demo

👉 https://moody-player-git-main-moody-players-projects.vercel.app/

---

## 🧠 Features

* 🎭 Real-time **Face Expression Detection** (face-api.js)
* 🎵 **Mood-based song recommendations**
* 📂 Upload songs with mood tagging
* ☁️ Audio storage using **ImageKit**
* 🌐 Fully deployed (Frontend + Backend)
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Face-api.js
* Axios
* CSS

### Backend

* Node.js
* Express.js
* MongoDB (Atlas)
* Multer (file upload)
* ImageKit (audio storage)

### Deployment

* Frontend → Vercel
* Backend → Render

---

## ⚙️ How It Works

1. User opens the app
2. Camera detects face
3. Mood is analyzed (happy, sad, neutral, etc.)
4. Frontend sends mood to backend
5. Backend fetches songs from MongoDB
6. Songs are displayed 🎵

## 🔧 Installation (Local Setup)

### 1. Clone repo

```bash
git clone https://github.com/harshpal130/Moody-Player.git
cd Moody-Player
```

---

### 2. Setup Backend

```bash
cd backend
npm install
```


Run backend:

```bash
node server.js
```

---

### 3. Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
